# AlkabotDesafio

## Descrição do desafio
### Bloggin é uma plataforma desenvolvida para um desafio. Tem como principal objetivo ver as postagens feitas, os comentários associados e uma listagem de usuários com suas informações.

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out the [documentation](https://nuxtjs.org).
